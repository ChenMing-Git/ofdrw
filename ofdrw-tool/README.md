# OFD Reader & Writer 文档操作工具

## 文档合并

1. 创建空文档
2. 输入待合并文档
3. 选择页面
   - 资源分析
   - 资源复制
   - 页面复制
   - 页面资源替换
4. 合并成文件


页面涉及资源：

- 页面模板
- 绘制参数 （DrawParam）
- 颜色空间
- 字体
- 多媒体
- 矢量图像


- 注释

相同资源

- 资源对象
- 资源文件

相同文件 关联到不同页码


每个文档 资源映射表

ID -> 资源对象 | 关联文件 (Hash) -> 新文档ID 

查表流程
```
原文档ID -> 查表 -> 新ID
            -> 对象关联文件 -> 计算文件Hash -> 查文件表 -> 存储
```
